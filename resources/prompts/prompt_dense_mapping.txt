Your job is to map a script to a DENSE sequence of visual anchors.

### CONTEXT:
The user wants a "High-Retention" style where images change frequently to keep the viewer engaged. 
Sometimes there are multiple images for a single sentence.

### TOOLS (PROVIDERS):
{providers_manifest}

### INSTRUCTIONS:
1. **Read the Script and the Vibe Analysis provided.**
2. **Create Visual Anchors:** 
    - Identify specific words or short phrases (anchors) that should trigger a new image.
    - Aim for high density: 1 image every 4-8 words for fast sections, 1 every 10-15 for slow ones.
3. **Select the Best Provider:** For EACH anchor, choose the most appropriate provider from the manifest.
4. **Generate Search Keywords:** Create 2-3 English search terms for that provider.

### CORE RULES:
- If the word is "Ancient Rome" → Use 'met' or 'pinterest' (diagrams).
- If the word is "Sadness" → Use 'unsplash' (moody photography).
- If the word is "How it works" → Use 'pinterest' (infographics).
- If the word is "Galaxy" → Use 'nasa'.
